
# EZD

This is a simple website which helps Drivers.




## Installation


```bash
git clone ssh-link-here
cd EZDrive
npm i express
npm i body-parser
npm i morgon
```


    
## Run Locally

Clone the project

```bash
  nodemon
```
the project will start running on port 8081 locally 
http://localhost:8081
